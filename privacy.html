Política de Privacidad

Última actualización: Noviembre 2024

1. Información del Responsable

Responsable del Tratamiento:
Angel

País: España

2. Datos que Recopilamos

FocusPro recopila los siguientes tipos de datos:

  • Datos de sesiones de enfoque (duración, tipo de tarea, configuraciones)
  • Estadísticas de productividad y métricas de rendimiento
  • Preferencias de usuario y configuraciones personalizadas
  • Información de cuenta (si el usuario crea una cuenta)
  • Datos de suscripción Premium (gestionados por RevenueCat)

3. Finalidad del Tratamiento

Utilizamos tus datos para:

  • Proporcionar las funcionalidades de la aplicación
  • Mejorar la experiencia del usuario
  • Analizar patrones de productividad
  • Gestionar suscripciones Premium

4. Base Legal

El tratamiento de tus datos se basa en:

  • Consentimiento del usuario: Al usar la aplicación, aceptas el tratamiento de tus datos.
  • Ejecución de contrato: Necesario para prestar el servicio de la aplicación.
  • Legítimo interés: Para mejorar y desarrollar el servicio.

5. Conservación de Datos

Los datos se conservan mientras el usuario tenga la aplicación instalada y activa. Los usuarios pueden solicitar la eliminación de sus datos en cualquier momento contactando con el email de soporte: angeldcchp94@gmail.com.

6. Derechos del Usuario

De acuerdo con el RGPD, tienes derecho a:

  • Acceso: Obtener información sobre tus datos personales
  • Rectificación: Corregir datos inexactos o incompletos
  • Supresión: Solicitar la eliminación de tus datos
  • Limitación: Restringir el tratamiento de tus datos
  • Oposición: Oponerte al tratamiento de tus datos
  • Portabilidad: Recibir tus datos en formato estructurado

Para ejercer estos derechos, contacta con: angeldcchp94@gmail.com

7. Transferencias Internacionales

Los datos pueden ser procesados por proveedores de servicios ubicados fuera del EEE (Espacio Económico Europeo), específicamente por:

  • Supabase: Almacenamiento de datos de sesiones y estadísticas
  • RevenueCat: Gestión de suscripciones Premium
  • Apple/Google: Procesamiento de pagos y datos de cuenta

Todos estos proveedores cumplen con las garantías adecuadas según el RGPD.

8. Seguridad de los Datos

Implementamos medidas de seguridad técnicas y organizativas para proteger tus datos:

  • Conexiones encriptadas (HTTPS/TLS)
  • Protección de datos según estándares de seguridad de Supabase
  • Acceso restringido a datos personales

9. Cookies y Tecnologías Similares

La aplicación puede usar tecnologías de seguimiento para mejorar el servicio. El usuario puede gestionar estas preferencias en la configuración de la aplicación.

10. Edad Mínima

La aplicación está diseñada para usuarios mayores de 13 años. Si eres menor de edad, necesitas el consentimiento de tus padres o tutores legales para usar la aplicación.

11. Autoridad de Control

Tienes derecho a presentar una reclamación ante la Agencia Española de Protección de Datos (AEPD) si consideras que el tratamiento de tus datos personales vulnera el RGPD.
